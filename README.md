# WebGL2 기반 실시간 3D 행성계 시뮬레이션

---

## 주요 기능

### 1. 계층적 천체 변환 체계 
* **공전 및 동주기 자전 고증:** 부모 행성(지구)의 움직임 행렬을 자식 천체(달)가 그대로 상속받는 조인트 연산 구조를 기반으로 작동합니다.
* **지구 자전축 경사(23.5°) 구현:** 변환 행렬의 우항 결합 법칙을 활용하여 자전축을 Z축 기준 $23.5^\circ$ 먼저 기울인 후 공전 궤도 상에 올림으로써, 우주 공간 내에서 일정한 방향으로 기울어진 채 공전하는 현실적인 메커니즘을 재현했습니다.

### 2. 고급 조명 및 텍스처 파이프라인
* **Blinn-Phong 반사 모델:** 하프웨이(Halfway) 벡터를 도입하여 연산 효율을 높인 스펙큘러 하이라이트 공식을 셰이더에 구현했습니다.
* **비동기 텍스처 및 미니맵:** JavaScript `Promise` 기반 로더를 통해 대용량 천체 이미지를 비동기로 로드하며, GPU 내 선형 필터링(`GL_LINEAR`) 및 미니맵(`Mipmap`)을 생성하여 텍셀 깨짐을 최적화했습니다.

### 3. 인터랙티브 카메라 및 사용자 인터페이스 (UI/UX)
* **아크볼 회전 (Arcball Rotation):** 2D 마우스 드래그 좌표를 가상의 3D 반구 공간에 투영하고 외적축과 내적각을 추출하여 가시 공간을 직관적으로 회전하는 트래킹 기술을 적용했습니다.
* **마우스 휠 줌:** 카메라의 좌표를 실시간으로 변환하여 부드러운 줌인/줌아웃을 지원합니다.

---

## 조작 방법

* **마우스 드래그 (`Mouse Drag`):** 우주 공간 및 카메라 시점 아크볼 회전
* **마우스 휠 (`Mouse Wheel`):**  카메라 전진/후진
* **스페이스바 (`Spacebar`):** **[시간 정지]** 모든 천체의 공전 및 자전 애니메이션 정지 / 재개 (정지 상태에서도 아크볼 회전 및 줌 기능은 정상 가동)

---

## 사용 기술 및 개발 환경

* **Language:** JavaScript
* **Graphics API:** WebGL2 (OpenGL ES 3.0 기반 라이브러리 프리 순수 제어)
* **Helper Library:** `MV.js` (Matrix-Vector 연산 유틸리티), `initShaders2.js`
* **Assets:** Custom `.obj` 3D Mesh, 고해상도 천체 표면 텍스처 이미지

## 실행 방법
#### WebGL2의 보안 정책(CORS)상 내부 자원(.obj 파일 및 텍스처 이미지)을 올바르게 로드하려면 별도 설정이 필요합니다.
* [Windows] 크롬 바로가기 -> 복사 -> 이름 변경(아무거나) -> 우클릭 -> 속성 -> '대상'에 아래 명령어 **추가**
```
--disable-web-security --user-data-dir="C:\ChromeTemp"
```
